# Windows Quick Start Guide

На этой странице описано как быстро запустить проект на Windows.

## Требования

Перед началом убедитесь, что установлены:
- **Python 3.8+** - https://www.python.org/downloads/
- **Node.js 16+** - https://nodejs.org/
- **Git** (опционально) - https://git-scm.com/

После установки Python и Node.js не забудьте добавить их в переменную PATH Windows.

## Быстрый старт

### Шаг 1: Откройте Command Prompt или PowerShell

Перейдите в папку проекта:
```
cd path\to\project
```

### Шаг 2: Установка зависимостей

Запустите скрипт установки:

**Вариант А: Из командной строки**
```
scripts\install.bat
```

**Вариант Б: Двойной клик**
- Откройте папку `scripts`
- Дважды нажмите на `install.bat`

Скрипт установит все необходимые зависимости (может занять несколько минут).

### Шаг 3: Запуск сервера(ов)

Вы можете запустить сервера несколькими способами:

#### Вариант 1: Запустить оба сервера одновременно (Рекомендуется)
```
scripts\start_all.bat
```
или двойной клик по `start_all.bat` в папке `scripts`

Откроются два окна консоли - одно для фронта, одно для бэкенда.

#### Вариант 2: Запустить фронт и бэкенд отдельно

В отдельных окнах командной строки:

**Бэкенд (FastAPI):**
```
scripts\start_backend.bat
```
Будет доступен на http://localhost:8000

**Фронт (React/Vite):**
```
scripts\start_frontend.bat
```
Будет доступен на http://localhost:5173

## Доступ к приложению

После запуска оба сервера:

- **Приложение** → http://localhost:5173
- **API (Backend)** → http://localhost:8000
- **API документация** → http://localhost:8000/docs

## Остановка сервера

Просто закройте окно консоли (окна), в котором запущен сервер.

## Полезные команды

В папке `scripts`:
- `install.bat` - Установка всех зависимостей
- `start_all.bat` - Запуск фронта и бэкенда в отдельных окнах
- `start_frontend.bat` - Запуск только фронта
- `start_backend.bat` - Запуск только бэкенда

## Решение проблем

### "Python не найден" / "Node.js не найден"
1. Убедитесь, что Python и Node.js установлены
2. Добавьте их в PATH:
   - Python: Обычно `C:\Users\YourUsername\AppData\Local\Programs\Python\Python310\`
   - Node.js: Обычно `C:\Program Files\nodejs\`
3. Перезагрузите Command Prompt/PowerShell

### Порт уже используется
Если вы видите ошибку вроде "Address already in use", значит порт занят:
- Закройте другие экземпляры приложения
- Проверьте не запущено ли приложение в фоновом процессе (используйте Task Manager)

### Права доступа (Access Denied)
- Попробуйте запустить Command Prompt как Администратор
- Отключите временно антивирус
- Убедитесь, что папка проекта не защищена от записи
